# gaian

- Generative AI all knowlege.
- The goal is to make all findings available in a generative AI.
- GAIAN provides a mechanism for storing a variety of knowledge-based data in a database.
- In plain English, the langchain is wrapped and made available on the command line.
- Documentation is [here](https://hamacom2004jp.github.io/gaian/).


# Install

- Install gaian with the following command.

```bash
pip install gaian
gaian -v
```
